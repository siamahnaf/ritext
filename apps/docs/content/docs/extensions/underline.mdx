---
title: Underline
description: Toggle underline formatting on selected text.
---

The `Underline` extension adds underline formatting. It renders a button in the toolbar and supports `Ctrl+U` / `⌘+U`.

## Import

```tsx
import { Underline } from 'ritext/extension/underline';
```

## Usage

```tsx
const extensions = [
  // ...base extensions
  Underline,
];
```

## Configuration

```tsx
Underline.configure({
  tooltip: true,
  activeClassName: 'text-blue-500',
})
```
