---
title: Font Size
description: Change the font size of selected text.
---

The `FontSize` extension adds a dropdown to the toolbar for selecting the font size of selected text.

## Import

```tsx
import { FontSize } from 'ritext/extension/font-size';
```

## Requirements

Requires `TextStyle` from base extensions.

```tsx
import { TextStyle } from 'ritext/extension/base';

const extensions = [TextStyle, FontSize];
```

## Default Font Sizes

- Default (unset)
- 10px, 12px, 14px, 16px, 18px, 20px, 24px, 28px, 32px, 36px, 40px, 48px, 72px

## Custom Size List

```tsx
FontSize.configure({
  fontSizeList: [
    'Default',
    '12px',
    '14px',
    '16px',
    '18px',
    '24px',
    '32px',
    { label: 'Large', value: '48px' },
    { label: 'Reset', value: null },
  ],
})
```
