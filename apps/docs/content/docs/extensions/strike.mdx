---
title: Strike
description: Toggle strikethrough formatting on selected text.
---

The `Strike` extension adds strikethrough formatting. Supports `Ctrl+Shift+S` / `⌘+Shift+S`.

## Import

```tsx
import { Strike } from 'ritext/extension/strike';
```

## Usage

```tsx
const extensions = [Strike];
```
