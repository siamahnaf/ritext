---
title: Italic
description: Toggle italic formatting on selected text.
---

The `Italic` extension enables italic text formatting. It renders a button in the toolbar and supports `Ctrl+I` / `⌘+I`.

## Import

```tsx
import { Italic } from 'ritext/extension/italic';
```

## Usage

```tsx
const extensions = [
  // ...base extensions
  Italic,
];
```

## Configuration

```tsx
Italic.configure({
  tooltip: true,
  showInBubbleMenu: true,
})
```
