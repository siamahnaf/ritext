---
title: Line Height
description: Change the line height of selected paragraphs.
---

The `LineHeight` extension adds a dropdown for controlling the line height (line spacing) of text.

## Import

```tsx
import { LineHeight } from 'ritext/extension/lineheight';
```

## Requirements

Requires `TextStyle` from base extensions.

```tsx
import { TextStyle } from 'ritext/extension/base';

const extensions = [TextStyle, LineHeight];
```

## Usage

```tsx
const extensions = [TextStyle, LineHeight];
```

## Default Values

- Default (unset)
- 1.5
- 2
- 2.5
- 3
- 3.5
- 4

## Custom Line Heights

```tsx
LineHeight.configure({
  lineHeights: ['Default', '1.2', '1.5', '1.8', '2', '2.5'],
})
```
