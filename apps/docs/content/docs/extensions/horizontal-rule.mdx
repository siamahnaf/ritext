---
title: Horizontal Rule
description: Insert a horizontal divider line.
---

The `HorizontalRule` extension adds a button to insert a horizontal rule (`<hr>`) into the document.

## Import

```tsx
import { HorizontalRule } from 'ritext/extension/horizontalrule';
```

## Usage

```tsx
const extensions = [HorizontalRule];
```
