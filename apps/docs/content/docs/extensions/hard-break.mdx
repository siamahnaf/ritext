---
title: Hard Break
description: Insert a line break within a paragraph.
---

The `HardBreak` extension adds a button for inserting a hard line break (`<br>`) within a paragraph, without creating a new block.

## Import

```tsx
import { HardBreak } from 'ritext/extension/hardbreak';
```

## Usage

```tsx
const extensions = [HardBreak];
```
