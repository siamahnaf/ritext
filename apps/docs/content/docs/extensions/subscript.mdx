---
title: Subscript
description: Toggle subscript formatting on selected text.
---

The `Subscript` extension adds subscript text formatting (e.g., H₂O). Supports `Ctrl+,` / `⌘+,`.

## Import

```tsx
import { Subscript } from 'ritext/extension/subscript';
```

## Usage

```tsx
const extensions = [Subscript];
```

## Keyboard Shortcut

| Action | Windows / Linux | macOS |
|--------|----------------|-------|
| Toggle Subscript | `Ctrl+,` | `⌘+,` |
