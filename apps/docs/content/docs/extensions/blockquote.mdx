---
title: Block Quote
description: Format content as a block quote.
---

The `BlockQuote` extension adds support for block-level quotes, rendered with a left border accent.

## Import

```tsx
import { BlockQuote } from 'ritext/extension/blockquote';
```

## Usage

```tsx
const extensions = [BlockQuote];
```
