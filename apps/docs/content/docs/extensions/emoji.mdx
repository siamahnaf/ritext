---
title: Emoji
description: Insert emojis from a searchable picker.
---

The `Emoji` extension adds an emoji picker dropdown to the toolbar, allowing users to browse and insert emojis.

## Import

```tsx
import { Emoji } from 'ritext/extension/emoji';
```

## Usage

```tsx
const extensions = [Emoji];
```

## Configuration

```tsx
Emoji.configure({
  showInBubbleMenu: false,
  openOnClick: false,
})
```
