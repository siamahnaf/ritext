---
title: Ordered List
description: Create numbered (ordered) lists.
---

The `OrderedList` extension adds support for numbered lists. Requires `ListItem` from base extensions.

## Import

```tsx
import { OrderedList } from 'ritext/extension/orderedlist';
```

## Requirements

```tsx
import { ListItem } from 'ritext/extension/base';

const extensions = [ListItem, OrderedList];
```

## Usage

```tsx
const extensions = [ListItem, OrderedList];
```
