---
title: Bold
description: Toggle bold formatting on selected text.
---

The `Bold` extension enables bold text formatting. It renders a button in the toolbar and supports the keyboard shortcut `Ctrl+B` / `⌘+B`.

## Import

```tsx
import { Bold } from 'ritext/extension/bold';
```

## Usage

```tsx
const extensions = [
  // ...base extensions
  Bold,
];
```

## Configuration

```tsx
Bold.configure({
  tooltip: true,
  activeClassName: 'bg-gray-200',
})
```
